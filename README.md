# Youtalk — Блог о психотерапии и самопомощи

Адаптивный сайт-блог для компании YouTalk, специализирующейся на психотерапии, самопомощи, отношениях и личных границах.

## 📋 Описание проекта

Проект представляет собой двухстраничный сайт:
- **Главная страница (index.html)** — блог со списком статей, фильтрацией по тегам и кнопкой "Ещё" для загрузки дополнительных материалов.
- **Страница статьи (article.html)** — детальный просмотр статьи с оглавлением, цитатами, списками и боковой панелью (промо-карточка, карточка психолога, баннер, пакет услуг).

## 🛠 Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, медиа-запросы)
- JavaScript (бургер-меню, подписка, загрузка статей)
- БЭМ-методология
- Figma (дизайн-макет)

## 🎯 Функциональность

- Адаптивная вёрстка под десктоп, планшет и мобильные устройства
- Бургер-меню на мобильных экранах
- Форма подписки на новости
- Фильтрация статей по тегам (демо-режим)
- Динамическая подгрузка статей по кнопке "Ещё"
